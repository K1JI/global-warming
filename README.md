🌍 Итоговый проект — «Глобальное потепление»
———————————————————————————————————————————————————————————————————
📌 Описание проекта:
  Данный проект представляет собой Telegram Mini App, посвящённый теме глобального потепления и экологии.
  Цель проекта — повысить осознанность пользователей, показать влияние человека на окружающую среду и предложить 
  интерактивный способ обучения через игру и взаимодействие с ботом.
  Проект разработан в рамках изучения языка программирования Python.
———————————————————————————————————————————————————————————————————
🧠 Используемые технологии:
  Python (Flask) — основной язык и «мозг» проекта.
  Вся логика, расчёты, работа с базой данных и API реализованы на Python.
  SQLite — база данных проекта.
  Используется для хранения информации о пользователях, результатах, очках и прогрессе.
  Telegram Bot API — взаимодействие с пользователем через Telegram-бота.
  HTML — структура веб-приложения (Mini App).
  CSS — оформление и дизайн интерфейса.
  JavaScript — используется минимально, только для связи Web-приложения с Telegram (Telegram Mini App API).
———————————————————————————————————————————————————————————————————
🎮 Функциональные возможности проекта
🌱 Калькулятор экологического / углеродного следа
🧩 Мини-игра на экологическую тематику
🏆 Система очков и результатов
🗄 Хранение данных пользователей в базе данных
🤖 Telegram-бот с командами и меню
🔊 Озвучивание текста (Text-to-Speech)
🎙 Распознавание голосовых сообщений (Speech-to-Text)

🎯 Цель проекта:
